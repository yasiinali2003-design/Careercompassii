# Urakompassi Logo Files

Professional compass-based logo designs for Urakompassi brand.

## Logo Versions

### 1. `logo.svg` (Full Compass)
- **Size:** 200x200px
- **Use:** Primary logo, favicon, app icons
- **Features:** Complete compass rose with cardinal directions (N, S, E, W)
- **Colors:** Uses brand palette (#2B5F75, #E8994A, #4A7C59)

### 2. `logo-icon.svg` (Simplified Icon)
- **Size:** 120x120px
- **Use:** Small icons, favicons, mobile app icons
- **Features:** Simplified compass with N marker only
- **Colors:** Brand colors with clean minimal design

### 3. `logo-full.svg` (Logo + Text)
- **Size:** 300x80px
- **Use:** Header navigation, marketing materials
- **Features:** Compass icon + "Urakompassi" text
- **Includes:** Optional tagline "Urasuunnittelu oppilaille"

### 4. `logo-dark.svg` (Dark Background Version)
- **Size:** 200x200px
- **Use:** Light backgrounds, dark themes
- **Features:** Dark background (#1F4756) with amber accents
- **Colors:** Inverted color scheme for contrast

### 5. `logo-simple.svg` (Minimalist)
- **Size:** 120x120px
- **Use:** When minimal design is needed
- **Features:** Clean, simple compass without decorative elements
- **Colors:** Core brand colors only

## Brand Colors Used

- **Primary:** #2B5F75 (Deep Teal-Blue)
- **Secondary:** #E8994A (Warm Amber)
- **Accent:** #4A7C59 (Forest Green)
- **Background:** #F8FAFB (Cool White)
- **Dark:** #1F4756 (Dark Teal)
- **Error/Accent:** #D64545 (Red for North needle)

## Usage Guidelines

1. **Minimum Size:** Logo should not be displayed smaller than 40px height
2. **Clear Space:** Maintain at least 20% of logo height as clear space around it
3. **Background:** Use `logo-dark.svg` on light/colored backgrounds
4. **Favicon:** Use `logo-icon.svg` at 32x32px or 64x64px
5. **Print:** Use SVG for scalability, or export PNG at 2x resolution

## Design Philosophy

The logo incorporates:
- **Compass Rose:** Represents navigation and guidance (core to career guidance)
- **Cardinal Directions:** Symbolizes finding your path
- **Professional Colors:** Trustworthy Nordic palette
- **Hand-crafted Look:** Custom-designed, not template-based

## Export Formats

If you need other formats:
- **PNG:** Export from SVG at 2x resolution (e.g., 400x400px for 200x200px logo)
- **PDF:** Use for print materials
- **ICO:** Convert for Windows favicons
- **Apple Touch Icon:** Use logo-icon.svg at 180x180px

## Customization

All logos are SVG format and can be edited:
- Colors can be adjusted in the SVG code
- Sizes are scalable (viewBox maintains aspect ratio)
- Text can be modified in `logo-full.svg`
